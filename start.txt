五、直接训练（不通过 Agent，使用 GPU）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    D:\LoveFlow\.venv\Scripts\python D:\LoveFlow\backend\run_loveflow_train.py --epochs 5

参数说明:
    --epochs                   训练轮数（默认 5）
    --lr                       学习率（默认 0.0002）
    --lora_r                   LoRA 秩（默认 8）
    --batch_size               批次大小（GPU 推荐 4-8，默认 4）
    --gradient_accumulation    梯度累积步数（默认 2）
    --max_seq_length           最大序列长度（默认 1024）
    --data_source              数据来源（mysql 或 web）

示例:
    D:\LoveFlow\.venv\Scripts\python D:\LoveFlow\backend\run_loveflow_train.py --epochs 10 --batch_size 8
    D:\LoveFlow\.venv\Scripts\python D:\LoveFlow\backend\run_loveflow_train.py --epochs 3 --lr 1e-4 --lora_r 16

MYSQL_USER                MySQL 用户（默认 root）
MYSQL_PASSWORD            MySQL 密码（默认 root）

七、常用工作流
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

完整流程（从采集到训练）:
    1. python -m agent.main
       → 进入交互对话
    2. "搜索关于最新 AI 技术的资料"
       → Agent 搜索互联网，爬取内容，生成训练数据
    3. "用这些数据训练 5 个 epoch"
       → Agent 启动 GPU 训练
    4. /status
       → 查看训练进度
    5. 训练完成后重启推理服务
       → python D:\LoveFlow\backend\run_loveflow.py
    6. 对话测试新模型
       → 在 chat 中直接对话

快速训练（已有数据）:
    D:\LoveFlow\.venv\Scripts\python -m agent.main train --epochs 10
